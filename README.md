# 명령어 정리
>#vue cli - https://cli.vuejs.org/
```
#19/12/22 version::  + @vue/cli@4.1.
npm install -g @vue/cli 
```

```
vue create '프로젝트명' #프로젝트 생성
-> default 선택
cd '프로젝트명'
npm run serve #package.json script 명령어 서버 실행 
npm i axios #axios 설치
```
